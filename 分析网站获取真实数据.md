# 如何获取 pfsc.agri.cn 真实数据

## 问题分析

网站返回401错误，说明：
1. API需要认证token
2. 或者API接口地址不对
3. 或者网站使用JavaScript动态加载数据

## 解决方案

### 方案1：使用Selenium（推荐）

网站可能是SPA应用，数据通过JavaScript动态加载。

**步骤1：安装Selenium**
```bash
pip install selenium webdriver-manager
```

**步骤2：下载Chrome驱动**
- 访问：https://chromedriver.chromium.org/
- 下载对应Chrome版本的驱动
- 或者使用webdriver-manager自动管理

**步骤3：修改爬虫使用Selenium**

我已经创建了 `pfsc_selenium_spider.py`，会自动尝试使用。

### 方案2：分析网站实际API

**步骤1：打开浏览器开发者工具**
1. 访问 https://pfsc.agri.cn/#/priceMarket
2. 按F12打开开发者工具
3. 切换到"Network"（网络）标签
4. 刷新页面

**步骤2：查找真实API请求**
- 筛选XHR或Fetch请求
- 查看请求URL、请求头、请求参数
- 查看响应数据格式

**步骤3：复制请求信息**
- 找到获取价格数据的API
- 复制请求URL、Headers、参数
- 修改爬虫代码使用这些信息

### 方案3：手动分析网站结构

1. 查看页面源码（Ctrl+U）
2. 查找数据相关的JavaScript变量
3. 查找内嵌的JSON数据
4. 分析数据加载逻辑

## 快速测试

运行以下命令测试Selenium爬虫：

```bash
# 安装依赖
pip install selenium webdriver-manager

# 运行爬虫
python run_spider_test.py
```

如果Selenium可用，会自动使用Selenium爬虫。

## 如果还是无法获取数据

可能需要：
1. 网站需要登录 - 需要处理登录逻辑
2. 网站有反爬虫机制 - 需要更复杂的处理
3. 网站使用加密 - 需要逆向分析

建议：
- 先使用测试数据完成系统开发
- 后续再针对实际网站进行深度分析
- 或者寻找其他公开的农产品价格数据源
