# Django Admin 后台登录说明

## 一、创建超级用户

系统**没有默认的用户名和密码**，需要先创建超级用户才能登录。

### 创建步骤

1. **激活虚拟环境**（如果使用虚拟环境）
   ```bash
   # Windows
   .venv\Scripts\activate
   
   # Linux/macOS
   source .venv/bin/activate
   ```

2. **执行创建命令**
   ```bash
   python manage.py createsuperuser
   ```

3. **按提示输入信息**
   ```
   用户名 (leave blank to use 'your_username'): admin
   电子邮件地址: admin@example.com
   Password: [输入密码，不会显示]
   Password (again): [再次输入密码确认]
   ```

   **注意**：
   - 密码输入时不会显示任何字符（这是正常的安全行为）
   - 直接输入即可，输入完成后按回车
   - 两次输入的密码必须一致

4. **登录后台**
   - 访问：`http://localhost:8000/admin/`
   - 使用刚才创建的用户名和密码登录

## 二、如果忘记密码

### 方法一：重置密码（推荐）

```bash
python manage.py changepassword 用户名
```

例如：
```bash
python manage.py changepassword admin
```

然后按提示输入新密码。

### 方法二：使用Django Shell重置

```bash
python manage.py shell
```

在Python shell中执行：
```python
from django.contrib.auth.models import User
user = User.objects.get(username='admin')  # 替换为你的用户名
user.set_password('新密码')
user.save()
exit()
```

### 方法三：创建新超级用户

如果忘记了用户名，可以创建一个新的：
```bash
python manage.py createsuperuser
```

## 三、查看现有用户

### 使用Django Shell查看

```bash
python manage.py shell
```

```python
from django.contrib.auth.models import User

# 查看所有用户
users = User.objects.all()
for user in users:
    print(f"用户名: {user.username}, 邮箱: {user.email}, 是否超级用户: {user.is_superuser}")

# 查看所有超级用户
superusers = User.objects.filter(is_superuser=True)
for user in superusers:
    print(f"超级用户: {user.username}")
```

## 四、常见问题

### Q1: 输入密码时看不到字符？

**A:** 这是正常的安全行为。在终端中输入密码时，为了安全起见，系统不会显示任何字符（包括星号）。直接输入密码后按回车即可。

### Q2: 提示"密码太简单"？

**A:** Django默认要求密码不能太简单。请使用：
- 至少8个字符
- 包含字母和数字
- 避免使用常见密码（如123456、password等）

### Q3: 登录后看不到数据？

**A:** 检查：
1. 是否已执行数据库迁移：`python manage.py migrate`
2. 是否已创建数据源并运行爬虫
3. 检查用户权限（确保是超级用户）

### Q4: 如何创建普通用户？

**A:** 在Django Admin中：
1. 登录后台
2. 进入"用户"（Users）
3. 点击"添加用户"（Add user）
4. 填写用户名和密码
5. 保存后可以编辑用户详细信息

## 五、快速创建测试用户

如果需要快速创建一个测试用的超级用户，可以使用以下命令（**仅用于开发环境**）：

```bash
python manage.py shell
```

```python
from django.contrib.auth.models import User

# 创建超级用户（如果不存在）
if not User.objects.filter(username='admin').exists():
    User.objects.create_superuser(
        username='admin',
        email='admin@example.com',
        password='admin123456'  # 请修改为更安全的密码
    )
    print("超级用户创建成功: admin / admin123456")
else:
    print("用户已存在")
```

**⚠️ 警告**：此方法仅用于开发测试，生产环境请使用 `createsuperuser` 命令并设置强密码。

## 六、登录地址

- **本地开发环境**：`http://localhost:8000/admin/` 或 `http://127.0.0.1:8000/admin/`
- **生产环境**：`http://your-domain.com/admin/`

## 七、安全建议

1. **使用强密码**：至少8位，包含大小写字母、数字和特殊字符
2. **定期更换密码**：建议每3-6个月更换一次
3. **不要共享账号**：为每个管理员创建独立账号
4. **生产环境**：修改默认的 `SECRET_KEY`，使用HTTPS

2. 数据库连接是否正常
