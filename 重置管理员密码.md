# 重置Django Admin管理员密码

## 方法一：查看现有用户并重置密码（推荐）

### 步骤1：查看所有用户

在项目根目录执行：

```bash
python manage.py shell
```

然后在Python shell中输入：

```python
from django.contrib.auth.models import User

# 查看所有用户
print("=" * 50)
print("所有用户列表：")
print("=" * 50)
users = User.objects.all()
for user in users:
    print(f"ID: {user.id}")
    print(f"用户名: {user.username}")
    print(f"邮箱: {user.email}")
    print(f"是否超级用户: {user.is_superuser}")
    print(f"是否激活: {user.is_active}")
    print("-" * 50)

# 只查看超级用户
print("\n超级用户列表：")
print("=" * 50)
superusers = User.objects.filter(is_superuser=True)
for user in superusers:
    print(f"用户名: {user.username}, 邮箱: {user.email}")
```

输入 `exit()` 退出shell。

### 步骤2：重置密码

找到用户名后，使用以下命令重置密码：

```bash
python manage.py changepassword 用户名
```

例如，如果用户名是 `admin`：
```bash
python manage.py changepassword admin
```

然后按提示输入新密码（输入时不会显示，直接输入后按回车）。

## 方法二：使用Django Shell直接重置

```bash
python manage.py shell
```

```python
from django.contrib.auth.models import User

# 方法1：如果知道用户名
username = 'admin'  # 替换为你的用户名
user = User.objects.get(username=username)
user.set_password('新密码123456')  # 替换为你想要的新密码
user.save()
print(f"用户 {username} 的密码已重置")

# 方法2：重置第一个超级用户的密码
superuser = User.objects.filter(is_superuser=True).first()
if superuser:
    superuser.set_password('新密码123456')  # 替换为你想要的新密码
    superuser.save()
    print(f"用户 {superuser.username} 的密码已重置为: 新密码123456")
else:
    print("没有找到超级用户")
```

输入 `exit()` 退出。

## 方法三：创建新的超级用户

如果忘记了用户名，可以直接创建一个新的超级用户：

```bash
python manage.py createsuperuser
```

按提示输入新的用户名、邮箱和密码。

## 方法四：一键重置脚本（快速方法）

创建一个Python脚本 `reset_admin_password.py`：

```python
#!/usr/bin/env python
"""
快速重置管理员密码脚本
"""
import os
import sys
import django

# 设置Django环境
os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'agricultural_platform.settings')
django.setup()

from django.contrib.auth.models import User

def reset_password():
    # 查找所有超级用户
    superusers = User.objects.filter(is_superuser=True)
    
    if not superusers.exists():
        print("没有找到超级用户，正在创建...")
        User.objects.create_superuser(
            username='admin',
            email='admin@example.com',
            password='admin123456'
        )
        print("已创建新超级用户:")
        print("  用户名: admin")
        print("  密码: admin123456")
        print("\n⚠️ 请尽快登录后修改密码！")
        return
    
    print("找到以下超级用户:")
    for i, user in enumerate(superusers, 1):
        print(f"{i}. {user.username} ({user.email})")
    
    if len(superusers) == 1:
        user = superusers.first()
        new_password = 'admin123456'  # 默认新密码
        user.set_password(new_password)
        user.save()
        print(f"\n✓ 已重置用户 '{user.username}' 的密码")
        print(f"  新密码: {new_password}")
        print("\n⚠️ 请尽快登录后修改密码！")
    else:
        print("\n请选择要重置的用户（输入序号）:")
        choice = input("> ").strip()
        try:
            index = int(choice) - 1
            if 0 <= index < len(superusers):
                user = superusers[index]
                new_password = 'admin123456'  # 默认新密码
                user.set_password(new_password)
                user.save()
                print(f"\n✓ 已重置用户 '{user.username}' 的密码")
                print(f"  新密码: {new_password}")
                print("\n⚠️ 请尽快登录后修改密码！")
            else:
                print("无效的选择")
        except ValueError:
            print("请输入有效的数字")

if __name__ == '__main__':
    reset_password()
```

然后运行：

```bash
python reset_admin_password.py
```

## 常见问题

### Q: 执行 `changepassword` 时提示用户不存在？

**A:** 先使用 `python manage.py shell` 查看所有用户，确认用户名是否正确。

### Q: 想查看所有用户的密码？

**A:** Django出于安全考虑，密码是加密存储的，无法直接查看。只能重置。

### Q: 如何删除旧用户？

**A:** 在Django Shell中：
```python
from django.contrib.auth.models import User
user = User.objects.get(username='旧用户名')
user.delete()
```

### Q: 如何批量重置所有超级用户密码？

**A:** 在Django Shell中：
```python
from django.contrib.auth.models import User
for user in User.objects.filter(is_superuser=True):
    user.set_password('新密码123456')
    user.save()
    print(f"已重置 {user.username} 的密码")
```

---

## 安全提示

1. **重置后立即登录修改密码**：使用强密码
2. **不要在生产环境使用简单密码**：如 `admin123456`
3. **定期更换密码**：建议每3-6个月更换
4. **使用HTTPS**：生产环境必须使用HTTPS保护登录
